# Axis

Declaring an axis constrains the element to move only in one direction, either horizontal or vertical.

<div v-drag="{ axis: 'x' }">Horizontal</div>

# Values

The axis has only two possible values:

Value Description
x Horizontal movement
y Veritcal movement
